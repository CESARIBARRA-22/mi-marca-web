# Chocofest - Tienda Oficial

Bienvenido al repositorio de **Chocofest**, la tienda oficial de chocolates y productos relacionados.

---

## 🔹 Descripción

Este proyecto contiene la página web de la tienda **Chocofest**, incluyendo:

- Página principal con catálogo de productos.
- Carrito de compras interactivo.
- Página de contacto con enlaces a redes sociales.
- Animaciones y slider automático de imágenes promocionales.
- Diseño responsive, adaptable a dispositivos móviles y desktop.

El objetivo es ofrecer una experiencia de compra sencilla y atractiva para los usuarios.

---

## 🔹 Tecnologías utilizadas

- **HTML5** para la estructura de la página.
- **CSS3** y **Google Fonts** para estilos y tipografía.
- **JavaScript** para interactividad, carrito de compras y sliders.
- **GitHub Pages** para publicar la página web en internet.
